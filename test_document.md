# Test Document for Second Brain

This is a sample document to test the ingestion functionality of the Fabric Second Brain system.

## Features

The Second Brain system can now:

1. **Process Documents**: Extract text from PDFs, Word documents, Excel spreadsheets, and more
2. **Handle Media**: Process audio and video files
3. **AI Summarization**: Automatically summarize content using Fabric AI
4. **Organize**: Automatically categorize content into appropriate vault folders

## Technical Details

The system supports the following document formats:
- PDF documents
- Microsoft Office (DOC, DOCX, XLS, XLSX, PPT, PPTX)
- OpenOffice formats (ODT, ODS, ODP)
- CSV and plain text files
- PostScript files (PS, EPS)

For media files:
- Video: MP4, AVI, MOV, MKV, WebM
- Audio: MP3, WAV, M4A, FLAC, OGG
- YouTube videos via URL

## Integration

All content is processed through the Fabric AI system and stored in your Obsidian vault with proper metadata, tags, and categorization.

This enables building a comprehensive knowledge management system that combines AI-powered content extraction with personal note-taking.
